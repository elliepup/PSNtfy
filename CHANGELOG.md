# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project uses Semantic Versioning.

## [0.1.0] - 2026-03-29

- Added a public-ready module manifest with project metadata.
- Hardened `Send-NtfyNotification` input validation and URI handling.
- Added support for common `ntfy` headers such as title, tags, priority, click URL, delay, filename, and Markdown mode.
- Returned structured response objects for easier automation.
